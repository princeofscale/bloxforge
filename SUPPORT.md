## Getting help

1. **Search issues** — check [existing issues](https://github.com/princeofscale/robloxstudio-mcp/issues) for solutions.
2. **Open a discussion** — for questions, workflow ideas, or help getting started, use [GitHub Discussions](https://github.com/princeofscale/robloxstudio-mcp/discussions).
3. **Open an issue** — for bug reports and feature requests, use the [issue tracker](https://github.com/princeofscale/robloxstudio-mcp/issues).

Before opening a bug report:
- Run `npx -y @princeofscale/robloxstudio-mcp@latest --doctor` and include the output.
- Check [Allow HTTP Requests](docs/installation.md#prerequisites) is enabled.
- Fully restart Roblox Studio after plugin updates (plugins load at Studio startup only).
